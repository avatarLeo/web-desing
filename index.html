Começando no HTML

HTML (Linguagem de Marcação de HiperTexto) é usado para construir as paginas web, com o HTML nós vamos marcando os elementos através de tags, que deverá aparecer no nosso site na web. O HTML tem uma função semantica para o nosso navegador indicando os elementos como textos, links, imagens. Textos pedem ser marcado com a tag <p>, <img> para imagens e <h1> para titulos por exemplo. Os arquivos onde escrevemos os codigos HTML sempre irão terminar com .html indicando que aquele arquivo tem código HTML

Vamos conhecer a anatomia da tag

Uma tag em HTML vai começar com o sinal de menor que < e termina com o sinal de maior que > dentro da tag de abertura e fechameto vai o conteúdo da tag, que pode ser uma outra tag.

Tags podem ter atributos

Os atributos da tag são colocados depois do nome da tag. Os atributos servem para alterar o comportamento da tag ou fornecer metadados por exemplo <div class="principal" id="principal"> nesse caso class e id são o atributo e o que esta entre parentese o valor.

Anatomia do documento HTML

As principais tags

  • <!DOCTYPE html> — o doctype. É a parte inicial obrigatória do documento. Nas névoas do tempo, quando o HTML era novo (por volta de 1991/2), doctypes eram criados para agir como links para um conjunto de regras que a página HTML tinha que seguir para ser considerada um bom HTML, o que poderia significar checagem automática de erros e outras coisas úteis. No entanto, atualmente, eles não fazem muito sentido e são basicamente necessários apenas para garantir que o documento se comporte corretamente. Isso é tudo que você precisa saber agora.

  • <html></html> — o elemento <html>. Esse elemento envolve todo o conteúdo da página e às vezes é conhecido como o elemento raiz.

  • <head></head> — o elemento <head>. Esse elemento age como um recipiente de tudo o que você deseja incluir em uma página HTML que não é o conteúdo que você quer mostrar para quem vê sua página. Isso inclui coisas como palavras-chave e uma descrição que você quer que apareça nos resultados de busca, CSS para dar estilo ao conteúdo, declarações de conjuntos de caracteres e etc.

  • <meta charset="utf-8"> — esse elemento define o conjunto de caracteres que seu documento deve usar para o UTF-8, que inclui praticamente todos os caracteres da grande maioria dos idiomas escritos. Essencialmente, agora ele pode manipular qualquer conteúdo textual que você possa colocar. Não há razão para não definir isso e assim pode ajudar a evitar alguns problemas no futuro.

  • <title></title> — o elemento <title>. Ele define o título da sua página, que é o título que aparece na guia do navegador onde sua página é carregada. Ele também é usado para descrever a página quando você a adiciona aos favoritos.

  • <body></body> — o elemento <body>. Contém todo o conteúdo que você quer mostrar ao público que visita sua página, seja texto, imagens, vídeos, jogos, faixas de áudio reproduzíveis ou qualquer outra coisa.

Fundamentos de textos no HTML

Titulos

Existem seis elementos de título — <h1>, <h2>, <h3>, <h4>, <h5> e <h6> . Cada elemento representa um nível diferente de conteúdo no documento; <h1> representa o título principal, <h2> representa subtítulos, <h3> representa sub-subtítulos, e assim por diante.

Pargrafos

Os textos no nosso HTML são envolvidos com a tag <p> para indicar para o navegador que temos um paragrafo.

Listas

Muito do conteúdo da web é de listas e o HTML tem elementos especiais para elas. Listas de marcação sempre consistem em pelo menos 2 elementos. Os tipos mais comuns de lista são ordenadas e não ordenadas:

  1. Listas não ordenadas são para listas onde a ordem dos itens não importa, como uma lista de compras, por exemplo. Essas são envolvidas em um elemento <ul>.
  2. Listas Ordenadas são para listas onde a ordem dos itens importa, como uma receita. Essas são envolvidas em um elemento <ol>.
  3. Cada item dentro de uma lista deve ser envolvido por uma tag <li>

A tag <span>

O elemento <span>. Não tem semântica. Você usa-o para agrupar conteúdo quando deseja aplicar o CSS (ou fazer algo com JavaScript) sem dar-lhe nenhum significado extra

A tag <strong>

Para enfatizar palavras importantes, tendemos a enfatizá-las na linguagem falada e colocá-la em negrito na linguagem escrita. Para fazer isso no HTML temos a tag <Strong>
