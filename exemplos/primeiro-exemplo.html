O que é o HTML 5?

O Hypertext Markup Language, ou HTML 5 é uma linguagem de marcação, utilizada para criar sites na web junto com outras tecnologias, como o CSS (Cascading Style Sheets) e o JavaScript podemos criar blogs paginas de vendas e até mesmo rede sociais.

Pra que serve cada um deles?

  • HTML é responsavel pela estrutura do site.
  • CSS vai trabalhar o desing da pagina.
  • JavaScript é responsavel palas interações do site.
