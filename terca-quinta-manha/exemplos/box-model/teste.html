tudo no html é tratado como uma caixa

No HTML tudo é representado no navegador como uma caixa, caixas na vida real possuem uma altura largura e um espaço interno onde fica o conteúdo. Não é diferente no HTML vamos ter tudo isso e mais algumas coisas que veremos logo mais. No modelo de caixa temos 4 elementos: uma margem, borda, conteúdo e um espaço entre o conteúdo e a borda do elemento.

Nós podemos alterar todas essas propriedade do nosso elemento no CSS.

width para alterar a largura,

height para alterar a altura

margin para configurar o espaçamento esterno

padding para configurar o espaço entre o conteúdo e a borda.

border-width border-style border-color

podemos ter uma caixa dentro da outra, chamamos de alinhaento


caixas vam ter altura height e largura width


caixas vam ter bordas


temos preenchimento padding


tipos de caixas

box-level

cada caixa ocupa 100% da largua do viewport

exemplo: div, h1, p, main, header, nav, article, aside, footer, form, video

inline-level

a caixa ocupa apenas o conteudo e não quebra linha

exemplo: span, a, code, small strong, em, sup e sub, label, button, input, select
